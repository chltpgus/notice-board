# notice-board

notice-board는 로그인, 글 작성, 글 수정 등을 할 수 있는 게시판 사이트다.

# 제작 이유
모든 사이트의 기본 구성이라고 생각이 되는 로그인, 게시판을 만들어 실력 증진을 하고 싶었고 React, CRUD, REST API를 학습 및 사용하고 싶었다.

# 사용 기술 
Netlify(웹호스팅), React, Html, Css, Javascript
